How to Calculate Deck Lumber

Calculate deck area (length × width), then divide by the board width to get the number of rows needed. For joists, divide deck length by joist spacing and add 1 for the end joist.

Pro tip: Use pressure-treated lumber (rated for ground contact) for the frame. Composite decking costs more upfront but requires far less maintenance over time.
